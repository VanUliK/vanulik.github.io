# Как обновить данные для нового чемпионата

## Что нужно сделать:

### 1. Обновить файл `data/users.json`
Замените фамилии участников на новые.

### 2. Обновить файл `data/matches.json`
Замените названия матчей на новые. 
**Важно:** сохраняйте структуру с этапами (как в примере).

### 3. Обновить файл `data/predictions.json`
Вставьте новые прогнозы в том же порядке, что и матчи.

### 4. Обновить файл `data/real-scores.json`
Вставьте реальные счета после того, как матчи сыграны.

## Полезные советы:

## Полезные советы:

1. **Структура этапов** должна сохраняться:
   - ГРУППОВОЙ ЭТАП (все матчи групп)
   - 1/16 ФИНАЛА
   - 1/8 ФИНАЛА
   - 1/4 ФИНАЛА
   - 1/2 ФИНАЛА
   - ЗА 3-е МЕСТО
   - ФИНАЛ
   - ЧЕМПИОН (специальный этап для победителя)

2. **Формат счетов**:
   - Для обычных матчей: "2:0" или "2-0" (оба варианта работают)
   - Для матча "Кто победит?": пишите название команды (например, "Испания")

3. **Количество прогнозов**:
   - У каждого пользователя должно быть столько же прогнозов, сколько матчей
   - Если прогнозов меньше - будут ошибки

4. **Пустые ячейки**:
   - Можно оставлять пустые строки "" - они будут отображаться как "—"

## Быстрый чек-лист перед запуском:

- [ ] Все JSON файлы в папке `data/`
- [ ] Файлы называются правильно: `matches.json`, `predictions.json`, `users.json`, `real-scores.json`
- [ ] В `users.json` перечислены все участники
- [ ] В `matches.json` есть все матчи и этапы
- [ ] В `predictions.json` у каждого пользователя столько же прогнозов, сколько матчей
- [ ] В `real-scores.json` столько же записей, сколько матчей
